Cool YouTube Player is a really cool player for YouTube videos.

== Description ==
Cool YouTube Player is a really cool player for YouTube videos. This plugin prevents to copy YouTube URL from your video. Also prevents the user to get redirect to YouTube site.

== ShortCodes ==
You have to use the following shortcut:
[coolyt videoId=b6KVRoHbH1w]
videoId should be filled with the Youtube Video ID.
